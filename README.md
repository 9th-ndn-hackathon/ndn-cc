Dependencies
============
- Python 3.6
- NodeJS (optional)

Setup Development Environment
=============================
Python venv:
```bash
python3 -m venv ./venv
./venv/bin/python -m pip install -r requirements.txt
```

Electron:
```bash
npm install
```

Execute
=======
Electron:
```bash
npm start
```

Python server:
```bash
./venv/bin/python app.py
```
